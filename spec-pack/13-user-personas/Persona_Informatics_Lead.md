# Persona: Informatics Lead

**Name**: Dr. Aisha Kumar (fictional exemplar)  
**Role/Job Title**: Informatics Lead  
**Workplace/Organization**: Clinical Society / Publisher  
**Location**: London, UK

---

## Basic Information

- Age: 35--55
- Education/Certifications: MD or PharmD, MSc/PhD in Health Informatics
- Years of Experience: 10--20 in informatics and clinical guideline modeling

## Demographics

- Specialty: Health informatics, guidelines, interoperability standards
- Informatics Experience: Expert—SNOMED CT, LOINC, RxNorm, HL7 FHIR, CDS integrations
- Technical Skills: Model computable guidelines, semantic mapping, standards harmonization, value set management
- Typical Tasks: Export computable guidelines, select code systems, align models with EHR/CDS standards

## Background/Expertise

- Primary Goals: Deliver computable, integrable guidance for CDS and EHR integration
- Motivations: Improve semantic interoperability, accelerate evidence-based care

## Goals/Motivations

- Needs: Authoritative mappings, SNOMED/LOINC/RxNorm exports, access to value set repositories
- Pain Points: Complex code alignment, rapidly changing standards, incomplete mapping documentation

## Activities

- Curates value sets, generates computable exports, validates model and mapping logic, manages updates
- Collaboration: Works closely with knowledge editors, terminologists, CDS architects, product managers
- Tools: FHIR Shorthand, VSAC, code browsers, mapping platforms

## Workflows/Behaviors

- Proficiency: High in computable modeling, moderate in consensus-building and clinical updates
- Support Needed: Up-to-date code systems, mapping workspaces, feedback channels for model integration

## Technology Comfort

> "Clinical knowledge only changes care when it’s mapped, coded, and engineered for real-world interoperability."
