# BDD Test Examples

This folder contains example BDD test scenarios demonstrating the Clinical BDD Creator system.

## Organization

- `by-domain/` - Examples by clinical specialty
- `by-mode/` - Examples by generation strategy  
- `by-fidelity/` - Examples by output detail level

## File Naming

Pattern: `{domain}-{mode}-{fidelity}-{scenario-id}.feature`

Example: `cardiology-bottom-up-full-chf-001.feature`

## Getting Started

Add your example BDD test files here! The system will use these for validation and demonstration.
